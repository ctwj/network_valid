## ADDED Requirements

### Requirement: 默认登录规则下的认证流程

SDK SHALL 支持默认登录规则下的认证流程：
- 普通登录模式：登录时不强制绑定设备，支持多设备同时在线
- 普通注册模式：注册无需激活码，用户名密码即可注册
- 任意解绑模式：解绑无次数限制，不扣时长

#### Scenario: 默认规则下的用户注册
- **WHEN** 用户在默认规则项目下注册
- **THEN** 仅需用户名和密码即可完成注册

#### Scenario: 默认规则下的用户登录
- **WHEN** 用户在默认规则项目下登录
- **THEN** 登录成功后不强制绑定设备，可多设备同时在线

#### Scenario: 默认规则下的用户解绑
- **WHEN** 用户在默认规则项目下解绑设备
- **THEN** 解绑成功且不扣除时长
