## Why

当前系统缺少完整的 API 对接文档，第三方开发者难以快速接入会员验证系统。需要基于现有 API 接口代码和 Apipost 文档，产出一份清晰的对接文档，包含接口说明、请求参数、响应格式、错误码等。

## What Changes

- 新增 `API对接文档.md`，包含所有对外 API 接口的完整说明
- 文档涵盖：时间戳接口、获取权限、获取软件信息、验证码、用户注册/登录/解绑/充值/心跳/下线等接口
- 包含请求参数说明、响应格式示例、错误码对照表

## Capabilities

### New Capabilities

- `api-integration`: API 对接文档，包含所有对外接口的完整说明和对接指南

### Modified Capabilities

无需求变更，仅添加文档

## Impact

- **文档**: 新增 `API对接文档.md`
- **代码**: 无变更
